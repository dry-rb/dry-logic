## Respond to

> See `attr?`.

``` ruby
respond_to?(:name)
```

``` ruby
# Pass
Struct.new(:name).new("John")

# Fail
Struct.new(:age).new(100)
```

## URI

> Verify user input is a URL.

``` ruby
uri?(:https)
uri?(:http)
uri?(/https?/)
uri?([:http, :https])
```

``` ruby
# Pass
"https://google.com"

# Fail
"localhost"
```

## UUID (1-5)

> Verify user input against UUID 1-5

``` ruby
uuid_v1?
uuid_v2?
uuid_v3?
uuid_v4?
uuid_v5?
```

``` ruby
# Pass UUID-1
"554ef240-5433-11eb-ae93-0242ac130002"

# Fail
"localhost"
```

## Case expression

> Implements the `===` operator

``` ruby
case?(5..10)
case?(Integer)
```

``` ruby
# Pass
10
1

# Fail
100
"string"
```

## True / False

> Check for a boolean value

``` ruby

```

``` ruby
# Pass

false

# Fail

true
```

## Excluded values

## Attribute

> Checks if an object responds to a certain method. Uses `respond_to?` to check the input.

``` ruby

```

``` ruby
# Pass

# Fail
```


## Format

> Applies a regular expression to its input

``` ruby
format?(/^(A|B)$/)
```

``` ruby
# Pass
"A"
"B"

# Fail
"C"
"D"
```
